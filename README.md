# Snake 
### 程式介紹  
我們製作的是貪吃蛇遊戲，相信各位小時候都玩過，  
幫大家回味童年，是個簡單又好玩的經典趣味小遊戲。
### 程式規則  
防止蛇頭去撞到牆壁或吃到自己的身體。
### 程式玩法    
利用方向鍵控制貪吃蛇移動方向，想辦法吃到果實，  
蛇身將會隨之長大愈來愈長，
一旦失控撞上牆壁或自己的身體，遊戲就會結束。
